# Design and Analysis of Class EF2 Inverter

## Overview
This project focuses on the design, mathematical analysis, and simulation of a 500 W, 100 kHz Class EF2 inverter for high-frequency power conversion applications.

The Class EF2 topology combines the advantages of Class E and Class F inverters by incorporating a second-harmonic resonant network, enabling Zero-Voltage Switching (ZVS), reduced switching losses, and lower voltage stress across the switching device.

## Objectives
- Design a high-efficiency Class EF2 inverter.
- Derive analytical design equations for circuit parameters.
- Achieve Zero-Voltage Switching (ZVS).
- Validate inverter performance using PSIM simulation.
- Generate a regulated DC output using transformer and rectifier stages.

## System Specifications

| Parameter | Value |
|------------|---------|
| Output Power | 500 W |

## Methodology
1. Developed mathematical design equations for the Class EF2 topology.
2. Calculated resonant tank and harmonic tuning components.
3. Designed the inverter in PSIM.
4. Verified ZVS operation and switching waveforms.
5. Integrated transformer and rectifier stages for DC output generation.

## Key Features
- Zero-Voltage Switching (ZVS)
- Reduced switching losses
- Lower peak switch voltage stress
- High-frequency operation
- Transformer isolation and rectification stage
- Improved power conversion efficiency

## Results
- Successful implementation of a 500 W Class EF2 inverter.
- Verified soft-switching operation through PSIM simulation.
- Achieved reduced voltage stress compared to conventional Class E topology.
- Generated stable DC output through transformer-coupled rectification.
